# Philosophers

Este proyecto consiste en implementar el problema clásico de los filósofos comensales utilizando hilos y sincronización. El objetivo es gestionar correctamente el acceso concurrente a recursos compartidos (tenedores) para evitar condiciones de carrera y situaciones de interbloqueo (deadlock). El ejercicio pone a prueba los conocimientos sobre programación concurrente, manejo de procesos/hilos y sincronización mediante mutexes en C.

## Ejemplo de uso

./philosophers 5 800 200 200 7
